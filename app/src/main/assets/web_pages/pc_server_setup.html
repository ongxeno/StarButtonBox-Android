StarButtonBox PC Server Setup

Follow these steps to install and set up the StarButtonBox server application on your Windows PC:

  1. Download the Server Installer: Click the button below to download the StarButtonBoxServer_Installer_v1.0.exe file.

    Browser Warning: Your browser might show a warning when downloading executable (.exe) files. This is a standard security precaution. You may need to choose "Keep" or "Download anyway" to proceed.

    Download PC Server Installer
  2. Run the Installer: Once downloaded, locate StarButtonBoxServer_Installer_v1.0.exe and run it.

    Windows Warning: Windows Defender SmartScreen or other security software might display a warning like "Windows protected your PC". If this happens, you may need to click "More info" and then "Run anyway" to proceed with the installation. This is common for new applications that haven't built up a reputation yet.

    Follow the on-screen prompts to install the application (you can usually accept the default settings and installation location).
  3. Check Network: Ensure your PC (where you just installed the server) and your Android device are connected to the same Wi-Fi network. This is crucial for them to communicate.
  4. Launch and Start the Server: After installation, launch "StarButtonBox Server" from your Start Menu or Desktop shortcut. The server control panel window will appear.
    In the control panel, click the "Start Server" button.
  5. Allow Firewall Access: When you start the server for the first time (or after installation), Windows Defender Firewall might ask for permission. It's very important to:
    • Check the box for Private networks (and optionally Public networks if you understand the security implications for your network setup).
    • Click "Allow access".

    (If you miss this prompt or deny access, you may need to manually create a firewall rule later.)

  6. Return to the App: Once the server status in its control panel shows "Server Running" (or similar positive status), go back to your Android device and tap "Continue to Network Setup" in the StarButtonBox app.
